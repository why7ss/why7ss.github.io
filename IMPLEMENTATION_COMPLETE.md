DEBUGGING
==================================================

Если что-то не работает, проверьте:

1. Консоль браузера (F12):
   - Ищите сообщения вида "✅ Service Worker зарегистрирован"
   - "✅ Успешно подписались на push notifications"

2. Backend логи:
   - Ищите "✅ Push отправлена"
   - "❌ Ошибка отправки push"

3. DevTools → Application → Service Workers:
   - Должен быть зарегистрирован /sw.js
   - Статус "activated and running"

4. DevTools → Application → Manifest:
   - Проверьте webcongif

==================================================
FAQ
==================================================

❓ Не работает Web Push?
→ Проверьте, что добавили VAPID ключи в .env
→ Перезагрузите backend (npm run dev)
→ Очистите браузер (F12 → Application → Clear storage)

❓ Нету кнопки "Включить уведомления"?
→ Браузер может уже иметь разрешение или отклонение
→ Откройте DevTools → Settings → и найдите разрешения для localhost
→ Сбросьте разрешения

❓ Push приходит, но не как notification?
→ Убедитесь, что даны разрешения на notification
→ Проверьте, что Service Worker активен

❓ Как отписать пользователя от push?
→ Автоматически при удалении аккаунта
→ Или endpoint POST /api/users/push-unsubscribe
